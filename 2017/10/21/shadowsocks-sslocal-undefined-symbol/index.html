查阅相关资料

解决openssl升级到1.1.0后shadowsocks服务报错问题

主要原因： openssl 升级至 1.1.0 后，废弃了EVP_CIPHER_CTX_cleanup 函数，并用 EVP_CIPHER_CTX_reset 函数替代了EVP_CIPHER_CTX_cleanup 函数。
解决方法 ：替换程序中对 EVP_CIPHER_CTX_cleanup 的引用。

  1. 编辑 报错的文件，如上面示例中的

    /usr/lib/python2.7/site-packages/shadowsocks/crypto/openssl.py

  2. 将第52行

    libcrypto.EVP_CIPHER_CTX_cleanup.argtypes = (c_void_p,)

    改为

    libcrypto.EVP_CIPHER_CTX_reset.argtypes = (c_void_p,)`

  3. 将第111行

    ibcrypto.EVP_CIPHER_CTX_cleanup(self._ctx)

    改为

    libcrypto.EVP_CIPHER_CTX_reset(self._ctx)

  4. 保存文件
